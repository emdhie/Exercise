Table

 
Write a table program. The program should ask a user to input a number. This number is then used to print the table.

Examples:
     Input:
      Enter a number: 2
      Enter a number of multiples: 4
     Output: 2*1=2 2*2=4 2*3=6 2*4=8      Input:
      Enter a number: 2
      Enter a number of multiples: 3
     Output: 5*1=5 5*2=10 5*3=15
