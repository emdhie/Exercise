Ulam Sequence

 
A mathematician named Ulam proposed generating a sequence of numbers from any positive integer n (n > 0) is as follows: If n is 1, stop.
If n is even, the next number is n/2.
If n is odd, the next number is 3*n + 1.
Continue with this process until n reaches 1.

Example :
1.If user gives input integer 5 then Ulam Sequence will be in the below format:
16 -> 8 -> 4 -> 2 -> 1
2.If user gives input integer 4 then Ulam Sequence will be in the below format:
2 -> 1
Note : Make sure output format must be same as shown above.
