Talking Parrot

Problem: You recently bought a parrot who loves to talk - a lot. You decide to write a small program where you can enter what he talks and the program tells you the number of words spoken. When this exercise is run, it will ask you to enter a line of text. All the words from that line will be stored in an array of Strings called 'words'.
You have to print the length of the array to the system console.
What we expect:
If you entered the following line of text when the program was run:
"Hi there how are you"
We expect you to print the number '5' to the system console.
